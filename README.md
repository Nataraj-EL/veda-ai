# VedaAI – Intelligent Assignment Generator & Academic Paper Engine

VedaAI is a full-stack AI-powered academic content generation platform that transforms study materials into structured, exam-ready assignments and printable question papers with **LaTeX-powered mathematical rendering**, **AI evaluation pipelines**, and **production-grade PDF output formatting**.

It is designed for educators, students, and institutions to generate high-quality assessments instantly with professional formatting comparable to real examination boards.

---

## Key Features

### AI-Powered Assignment Generation
- Converts raw study material into structured assignments
- Supports MCQs, Short Answer, Long Answer, Numerical Problems, and Diagram-based questions
- Automatic difficulty classification and marking scheme generation

### Advanced AI Pipeline
- Multi-stage prompt engineering pipeline
- Gemini-based formatting refinement layer
- JSON validation + fallback recovery system
- GROQ-powered inference engine integration

### Professional PDF Output System
- A4-print optimized question paper layout
- Clean academic formatting with strict typography rules
- Dual-page structure:
  - Page 1: Question Paper
  - Page 2: Answer Key
- Custom footer system:
  - “Generated by VedaAI”
  - Dynamic page numbering per page

### LaTeX & KaTeX Rendering Engine
- Real-time mathematical rendering using KaTeX
- Supports:
  - Inline math
  - Block equations
  - Fractions, probabilities, statistics, ML formulas
- Sanitization layer to prevent malformed expressions

### Responsive UI/UX System
- Fully responsive mobile + desktop layouts
- Clean minimal academic interface
- Optimized spacing, typography, and hierarchy
- Unified search & filtering system for assignments

### Smart UX Enhancements
- Loading simulation pipeline for AI generation
- Real-time assignment status polling
- Graceful fallback handling for async generation delays
- Error-safe rendering system

### Production-Ready Architecture
- Type-safe full-stack TypeScript implementation
- Modular backend API structure
- Strict ESLint + build validation (0 errors)
- Environment-based configuration system
